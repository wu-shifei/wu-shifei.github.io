配置二级域名和DDNS

原本用的是华硕的免费ddns，但是跟一段时间需要重新申请，有一点麻烦。今天把自己原有的域名设置一个二级域名并设置DDNS。

在域名的DNS管理界面新增一条记录

类型：A
主机：二级域名前缀
指向：主机IP

DDNS设置

申请GoDaddy API Key

在API Key Management登录自己的账号并申请API Key


在2 API Key Details界面的信息只会显示一次，一定要记下来再推出。

ddns脚本执行

原本计划ddns有两个方案：

  1. 华硕路由器提供的免费ddns
    但是隔段时间要重新申请计较麻烦
  2. 软件中心（华硕路由器+Koolshare固件才有软件中心）的软件Gdddns
    但是执行有错误

由于上述2个问题，而且刚好之前在树莓派上跑了一个脚本来用邮件通知IP的变化，这一次索性就在次基础上在加一个ddns设定的脚本。
