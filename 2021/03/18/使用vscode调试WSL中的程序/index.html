使用vscode调试WSL中的程序

vscode扩展安装：

参考:https://docs.microsoft.com/zh-cn/windows/wsl/tutorials/wsl-vscode

  • 从Windows安装
  1. 安装Windows版vscode
  2. 在vscode中安装远程开发包[https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack]
  3. CTRL+SHIFT+P 在 VS Code 中打开命令面板。 如果随后键入，您 Remote-WSL 将看到可用的 VS Code 远程选项的列表，允许您在远程会话中重新打开该文件夹，指定要打开的分发，等等。
  • 从WSL安装
  1. 在WSL下执行
  2. 在WSL中的vscode中安装必要的扩展（C/C++等）
